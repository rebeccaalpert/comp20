Security Analysis of Dan Baigel's MapChat Application

Prepared by Rebecca Alpert

Introduction

This document assess the security of the MapChat web application developed by Dan Baigel for Comp 20 assignments 2 and 3.

The web application, according to the specs, retrieves and stores the current location and message of the client if the client's username is within an approved list of usernames, and then retreives and displays the locations and messages of each user on a map. The server possesses several functions: a POST /sendLocation API, a GET /latest.json API, and a homepage that displays the list of locations and messages for all logins.

I was assigned to identify at least three security or privacy vulnerabilities in the application.

Methodology

Initially, I performed black-box testing using curl.
